# Wärmepumpe CAN mit ESP8266 und MCP2515 auslesen

Connect to an Stiebel Eltron or tecalor Wärmepumpe / heat pump via an ESP8266.

## Hardware 

### MCP2515 - ESP8266

- MCP2515   -   ESP8266
- INT --        4  - D2
- SCK -- SCLK - 14 - D5 
-  SI -- MOSI - 13 - D7
-  SO -- MISO - 12 - D6
-  CS -- CS   - 15 - D8
- GND -- GND
- VCC -- 3,3V
